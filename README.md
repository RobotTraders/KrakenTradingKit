# KrakenTradingKit

Requires Python 3.12 or 3.13.

## Clone

```bash
git clone https://github.com/RobotTraders/KrakenTradingKit.git
cd KrakenTradingKit
```

PowerShell:

```powershell
git clone https://github.com/RobotTraders/KrakenTradingKit.git
cd KrakenTradingKit
```

## Install

```bash
uv sync
```

## Setup

Copy the example environment file and fill in your own API credentials:

```bash
cp .env.example .env
```
